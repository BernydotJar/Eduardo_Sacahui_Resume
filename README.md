# Eduardo Sacahui Resume Portfolio

Interactive resume and technical portfolio built with Next.js and deployed to GitHub Pages.

## Stack

- Next.js 15 (App Router)
- React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the app locally:

```bash
npm run dev
```

3. Open:

```text
http://localhost:9002
```

## Build

```bash
npm run build
```

This project uses static export (`output: "export"`) and publishes the generated `out/` folder via GitHub Actions.
