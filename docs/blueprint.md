# **App Name**: Sacahui Portfolio

## Core Features:

- Skills Grid: Render skills from JSON with hover effects, search/filter, and detail drawer integration.
- Signature Migrations: Render migration cards with outcomes chips.
- Experience Timeline: Create expandable timeline cards from experience data with tech chips.
- Contact Form: Implement contact form submission to Firebase Function and display toast notifications on success/failure.
- Periodic Table Easter Egg: Implement the 'Yeah, Science!' easter egg with tooltip text flips, tile glow, and accessibility toggle.
- AI-Powered SOP Guidance: Provide semantic SOP guidance leveraging OpenAI/Azure OpenAI for context-aware actions (Jira/ServiceNow) using the OpenAI tool.
- Code Generation From Schema: Leverage LLMs to generate API and data layer code from database schema.

## Style Guidelines:

- Primary color: Teal (#2DD4BF), inspired by the Breaking Bad/periodic table motif, providing a fresh, energetic feel.
- Background color: Dark slate (#0F172A), offering a sophisticated and modern backdrop that emphasizes content.
- Accent color: Emerald green (#34D399), used for hover states and highlights to provide interactive feedback.
- UI font: 'Inter' (sans-serif) for UI elements. Note: currently only Google Fonts are supported.
- Tile Symbol font: 'JetBrains Mono' (monospace) for the skill tile symbols.
- Use radioactive icons for education tiles to represent 'unstable isotopes'.
- Design tiles in the layout of the periodic table.
- Subtle pulsing animation for 'unstable isotope' education tiles.
